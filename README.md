## Usage

### Install
```bash
$ npm install -g vue-cli
# or
$ yarn global add vue-cli

$ vue init yuki-npm-template your-project-name
$ cd your-project-name

$ npm install
# or
$ yarn
```

### Publish

You can use `np`, or `npm publish`
```bash
$ npm install -g np
# or
$ yarn global add np

$ np
```
